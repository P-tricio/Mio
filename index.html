TECLAS:

  • Pulsa Q para tocar los platos.
  • Pulsa W para tocar el hi-hat cerrado.
  • Pulsa E para tocar el hi-hat abierto.
  • Pulsa A para tocar el bombo.
  • Pulsa S para tocar el ride.
  • Pulsa D para tocar la caja.
  • Pulsa Z para tocar el tom alto.
  • Pulsa X para tocar el tom medio.
  • Pulsa C para tocar el tom bajo.
